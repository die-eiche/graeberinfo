Grabbuch – Synology-Ablage (//synology/grabbuch/)
=================================================

Dieser Ordner wird beim Build nach "grabbuch/" kopiert.
Den kompletten Inhalt von "grabbuch/" auf die Synology legen:

  Ziel: \\synology\grabbuch\

Bauen:
  npm run build:grabbuch

Auf Synology kopieren (Windows):
  npm run sync:grabbuch

Oder manuell den Ordner "grabbuch" aus dem Projekt nach
\\synology\grabbuch\ kopieren.

Aufruf im Browser:
  http://<synology-ip>/grabbuch/
  http://<synology-ip>/grabbuch/Suche.html
  http://<synology-ip>/grabbuch/Suche.html?boden=1-boden.jpg

Grundriss-Bilder in grabbuch-static/bilder/ ablegen (werden mit deployed).
